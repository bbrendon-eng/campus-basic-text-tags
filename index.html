When you have text displayed in a block with multiple lines, it's considered a paragraph. Paragraph text may contain emphasized words or phrases and other markup such as <a> or <strong> or <em> to convey meaning.

Marking up Text in Blocks

It may happen that you have text, like a poem, that requires special formatting. You can use line break tags like <br> to ensure each line displays properly. These should not be used for layout; use CSS instead.
